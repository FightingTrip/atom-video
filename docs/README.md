## 快速开始

1. 克隆项目：
```bash
git clone https://github.com/your-username/atom-video.git
```

2. 安装依赖：
```bash
# 后端
cd backend
npm install

# 前端
cd frontend
npm install
```

3. 配置环境变量：
```bash
# 复制示例文件
cp .env.example .env

# 编辑 .env 文件，填写必要的配置
```

4. 启动开发服务器：
```bash
# 后端
cd backend
npm run dev

# 前端
cd frontend
npm run dev
```
