## 技术栈

- **核心框架**: Vue 3.x + TypeScript
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP 客户端**: Axios
- **CSS 方案**: 设计系统 + Tailwind CSS + 组件样式
- **测试框架**: Vitest + Vue Test Utils
